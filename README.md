# David Román — Personal Website

My personal website and portfolio, built to showcase my work and experience as a Frontend Developer.

🌐 [davidromanhernandez.com](https://davidromanhernandez.com)

## Tech Stack

- React
- TypeScript
- HTML5 / CSS3

## Getting Started

Clone the repo and install dependencies (node required):

npm install
npm run dev

Open http://localhost:3000 in your browser.
